# ⚡ Quick Reference Card

## 🔍 Essential Fuzzy Commands
```
Ctrl+R    → Search command history
Ctrl+T    → Find files  
Alt+C     → Navigate directories
fh        → Fuzzy history execute
fcd       → Fuzzy directory change
fe        → Fuzzy file edit
fkill     → Fuzzy process kill
fgco      → Fuzzy git branch checkout
```

## ⌨️ Kitty Shortcuts
```
F1        → Quick fuzzy history
F2        → Quick directory nav
F3        → Quick file edit
Ctrl+Shift+G → Git status
Ctrl+Shift+F → Search scrollback
Ctrl+Shift+T → New tab
Ctrl+Shift+F5 → Reload config
```

## 🛠️ Modern CLI Tools
```
ls        → exa (icons + git)
cat       → bat (syntax highlighting)  
grep      → ripgrep (faster)
find      → fd (simpler)
```

## 📝 Git Shortcuts
```
gs        → git status
ga        → git add
gc        → git commit
gp        → git push
gl        → git pull
gd        → git diff
gco       → git checkout
glog      → git log --graph
```
